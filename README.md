# Projeto Spinner 🔄

## 📌 Sobre o projeto
Projeto Spinner desenvolvido utilizando HTML e CSS, com foco em animações, efeitos visuais e criação de componentes interativos para carregamento de páginas.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 🎯 Objetivo
O objetivo deste projeto foi praticar:
- Animações com CSS
- Efeitos de carregamento
- Estilização de componentes
- Organização de código front-end
